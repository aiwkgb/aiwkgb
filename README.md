## Featured open-source project

### Indonesia AI Builder Starter Kit

[Indonesia AI Builder Starter Kit](https://github.com/aiwkgb/indonesia-ai-builder-starter-kit) is a public open-source starter kit for Indonesian beginner AI builders.

It includes practical docs, templates, checklists, examples, contributor guidance, and workflow references to help beginners learn how to plan, document, build, review, and maintain small AI-assisted projects.

The project is still early, but actively maintained through GitHub Issues, changelog, roadmap, and public releases.
